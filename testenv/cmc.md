ConfigServer MmmmmodSecurity Control (cmc)
==========================================

About
-----

This is a plugin for WHM that allows you to whitelist rules and work with ModSecurity.

Installation
------------

```bash
pushd /usr/local/src/
wget http://configserver.com/free/cmc.tgz
tar -xzf cmc.tgz
cd cmc/
sh install.sh
popd
```
